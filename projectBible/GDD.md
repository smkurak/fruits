# Fruit Garden — Game Design Document

## High Concept

Fruit Garden — уютная визуальная новелла о волшебном острове, где души фруктов воплощаются в девушек.

## Core Pillars

- Живые персонажи.
- Визуальная эволюция.
- Летняя атмосфера.
- Ботаническая эстетика.

## Игровой цикл

Утро → выбор локации → диалог/мини-игра → рост доверия → изменение персонажа → следующий день.
